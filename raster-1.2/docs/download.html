Download

Download raster-1.2.zip (~7MB). This archive contains the Raster's source, examples, and documentation.

ZIP Archive Contents:

  • /examples - Getting started examples.
  • /docs - Raster API documentation.
  • /photoshop - Raster Photoshop files and scripts to create sprite images. You can use these to create your own css color themes.
  • /raster - Raster redistributable files. Copy this directory to your web application root folder so your pages can link to Raster's scripts, css, and images.

Changes in version 1.2:

  • Patches for IE9, IE10, and IE11
  • Other minor bug fixes

Changes in version 1.1:

  • Added new RasterList.removeAll() method.
  • Bug fix: Could not call RasterTabItem.setEnabled() if no sprite icon is set in the tab.
  • Bug fix: Fixed error when calling RasterList.getItemByValue().
  • Bug fix: Sort arrow in list headers will shrink the header text when it appears, rather than overlay on top of the text.
  • Bug fix: Photoshop scripts were generating CSS without "px" unit.
  • Documented RasterControl.rasterControl property.
