RasterTabbar.setEventHandler( [eventHandler] )

Specifies an event handler to be notified of tab events.

Parameters

Name Type   Description
eventHandler function optional The event handler function to be invoked when a tab event occurs. Set to null to remove any previously set handler.

Returns

Type Description
object The tabbar object. This allow for chaining multiple setter methods in one single statement.
