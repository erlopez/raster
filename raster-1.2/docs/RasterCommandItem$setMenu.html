RasterCommandItem.setMenu( menu )

Assigns a menu to this command item.

Parameters

Name Type   Description
menu object A reference to a menu object. This is the popup menu shown then the command item is clicked. When a menu is specified, the drop down arrow is automatically turned on. Use showArrow() to turned off if desired.

Returns

Type Description
object The command item object. This allow for chaining multiple setter methods in one single statement.
