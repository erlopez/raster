RasterToolbar.cleanSeparators( )

Removes adjacent, trailing, or leading separators. This method is called after removing items from the control to make sure no adjacent separators are left when removing an item.

Returns

Type Description
object The toolbar object. This allow for chaining multiple setter methods in one single statement.
