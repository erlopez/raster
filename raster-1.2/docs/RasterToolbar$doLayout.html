RasterToolbar.doLayout( )

Show/hides the chevron menu based on the width of the toolbar. This is invoked automatically in page resize events, but can be called directly if a custom layout changes affects the width of a toolbar.

Returns

Type Description
object The toolbar object. This allow for chaining multiple setter methods in one single statement.
